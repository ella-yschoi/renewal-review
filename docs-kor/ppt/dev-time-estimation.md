# 순수 개발 시간 추정 — AI 없이 renewal-review 구축

## 목적

프레젠테이션용 비교 데이터. 동일 시스템을 AI agent 없이 혼자 만들었을 때의 시간 추정.

---

## 추정 근거

- 시니어 Python/FastAPI 개발자 기준
- 기존 보험 도메인 지식 없음 (학습 시간 포함)
- TDD 방식, 코드 리뷰 없이 solo 개발

---

## Phase별 추정

| Phase | 작업 | AI Agent | 순수 개발 | 비고 |
|-------|------|----------|-----------|------|
| 0. Scaffolding | 프로젝트 셋업, CI, linter | 10분 | 1시간 | uv init, pre-commit, ruff 설정 |
| 1A. Models + Parser | ACORD 모델 7개, 파서 | 30분 | 4시간 | ACORD 표준 리서치 + 모델 설계 |
| 1B. Diff Engine + Rules | 15종 규칙, differ | 45분 | 6시간 | 필드별 비교 로직, 경계값 테스트 |
| 1C. Mock Data Generator | 8,000건 생성기 | 20분 | 3시간 | 분포 설계, 엣지 케이스 |
| 1D. Batch + API | 배치 처리, FastAPI 라우트 | 30분 | 4시간 | async 처리, 에러 핸들링 |
| 2A. LLM Client + Prompts | 3종 프롬프트, Langfuse | 30분 | 5시간 | 프롬프트 엔지니어링 반복 |
| 2B. Analyzer + Aggregator | LLM 투입 판단, 결과 종합 | 30분 | 5시간 | 오케스트레이션 로직 |
| 2C. Eval + Migration | 골든셋, Basic/LLM 비교 | 20분 | 3시간 | eval 파이프라인 |
| Frontend | Jinja2 UI 7페이지 | 45분 | 6시간 | HTML/CSS, 인터랙션 |
| Testing | 100개 테스트 | 포함 | 포함 | 각 Phase에 포함 |

---

## 요약

| 지표 | AI Agent | 순수 개발 | 배율 |
|------|----------|-----------|------|
| **총 시간** | ~4시간 | ~37시간 | **~10x** |
| **작업 일수** | ½일 | 5일 (7.5h/일) | **10x** |
| **코드 라인** | ~2,500줄 | ~2,500줄 | 동일 |
| **테스트** | 100개 | 100개 | 동일 |

> 코어 시스템 반나절 + 5가지 실험 포함하여 총 ~2일 완료.

---

## 주요 시간 절약 포인트

1. **도메인 리서치 → 즉시 코드**: ACORD 표준, 보험 용어를 별도 학습 없이 바로 모델링
2. **보일러플레이트 제거**: Pydantic 모델, FastAPI 라우트, conftest 등 반복 코드 자동 생성
3. **프롬프트 엔지니어링 1회차**: 사람이라면 3-5회 반복할 프롬프트 설계를 구조화된 지시로 1회 완성
4. **테스트 병렬 작성**: 구현과 동시에 테스트 생성 (TDD 자동화)
5. **Mock 데이터 설계**: 분포 조건을 자연어로 지정 → 즉시 생성기 구현
